## How to run
Download the binary to your local dir, then just run 
```
$ ./sillybot
```

You can download the example file to your local, for example in the same directory of your binary, then just run
```
$ ./sillybot -f example.yaml
```

To get help, run
```
$ ./sillybot -h
```
